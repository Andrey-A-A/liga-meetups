Приложение MeetUps

Приложение MeetUps представляет из себя список мероприятий в котором вы можете найти для себя встречи по своим интересам и зарегистрироваться на участие в них. Также приложение позволяет создавать и редактировать планируемые встречи.

Это приложение учебное. При работе использовался следующий стек технологий: JavaScript, TypeScript, фреймворк Angular с встроенной библиотекой RxJS. RxJS — это библиотека, реализующая принципы реактивного программирования для JavaScript. Основанная на объектах типа Observable, она упрощает написание и контроль асинхронного и событийного кода. Для работы над стилями использовался препроцессор SCSS и библиотека Angular Material

Приложение состоит из следующих страниц:
  • стартовая страница с описанием приложения
  • общий список "все митапы"
  • список "мои митапы", состоящий из митапов созданных пользователем, а также из митапов на которые он подписан
  • список пользователей (доступен только администратору)
  • страница создания митапа
  • страница редактирования митапа
  • страница авторизации
  • страница регистрации
В приложении реализовано:
  • Поиск по всем митапам по названию, описанию и дате
  • В списке выводятся сокращенные версии митапов, которые можно развернуть, для получения подробной информации
  • В зависимости от даты проведения встречи выводится статус "проведено" или указывается место проведения, если встреча ещё предстоит.
  • В карточке митапа реализована возможность подписаться или отказаться от встречи
  • Реализована пагинация с выбором количества отображаемых карточек митапов на странице
  • В списке "мои митапы" для митапов, которые создал сам пользователь отображается кнопка для редактирования митапа
  • В форме редактирования митапа, форме создания митапа и форме регистрации пользователя используются реактивные формы с валидацией
  • В списке пользователей, который доступен только администратору, предусмотрено редактирование данных пользователя, с возможностью назначения ролей User/Admin
  • Ссылки в меню доступны только авторизованным пользователям, что достигается с помощью настроенных Route Guards, которые позволяют ограничить доступ к маршрутам на основе определенного условия
  • Форма для создания митапа доступна по нажатию знака шестиренки в панели меню
  • Без авторизации доступна только стартовая страница с описание приложения и собственно страница с авторизацией
